2. JavaScript Se puede integrar en HTML de 3 maneras

  1. La primera es usar etiquetas de Script en el body.

    2. body
    script
    Codigo js
    /script
    /body

  2. La otra forma es ponerlo en el head.

    3. head
    script
    Codigo js
    /script
    /head

  3. La tercera forma es insertandolo en el head pero agregandol una ruta al archivo de js

    4. head
    script src="archivo.js" /script
    /head
