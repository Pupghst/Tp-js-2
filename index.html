  1. Por declaracion

    2. Es la mas utilizada y mas facil de recordar , consiste en declarar la funcion con un nombre y sus parametros entre parentesis

    Ejmplo: function add(a, b){ return a+b;};add(5,6); //11

  2. Por expresion

    3. Esta forma a tomado popularidad y consiste en guardar una función en una variable, para así ejecutar la variable como si fuera una función

    Ejemplo: const plus = function (a, b){ return a+b;};plus(5,6); //11

  3. Por constructor de objeto

    4. La variante de constructor de objrto es la menos recomendada y no es utilizada mucho ,solo nos ayuda a recordar que las funciones también son tipos de objetos en JavaScript.
